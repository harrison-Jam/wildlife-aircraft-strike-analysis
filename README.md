# Wildlife Aircraft Strike Analysis (Tableau Dashboard)

## Project Overview
This project analyzes wildlife strike incidents involving aircraft using Tableau dashboards to identify operational risks, cost impact, and mitigation opportunities.

The analysis focuses on:
- Wildlife strike frequency
- Damage severity
- Operational risk factors
- Cost distribution
- Wildlife species impact

## Tools Used
- Tableau Desktop
- MySQL (Data Cleaning)
- GitHub

## Key Insights

1. Birds account for the majority of wildlife strikes and damage costs.
2. Approach and landing phases present the highest operational risk.
3. Most strikes occur below 1000 ft near airport runways.

---

## Recommendations

- Focus wildlife control at high-risk airports.
- Implement bird detection radar and migration monitoring.
- Strengthen low-altitude wildlife management.
- Improve airport perimeter fencing to prevent mammal incursions.
